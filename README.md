## 開発ルール <a id="development-rules"></a>

- **Git運用**
  - 1〜2時間ごとに Commit
  - 1日 2〜3回 Push
  - 作業前に必ず **Pull** して最新を取得
  - Commit メッセージは内容がわかる日本語 or 英語（例：`プレイヤー移動を追加`）

- **Unityプロジェクト設定**
  - Unity 6（URP）を使用
  - 不要アセットは増やしすぎない（軽量管理）
  - `.gitignore` は Unity 公式テンプレートを使用

- **フォルダ構成（推奨）**
```
Assets/
├─ Scripts/
├─ Prefabs/
├─ Scenes/
├─ UI/
└─ Audio/
```
